Componentes Utilizados

  • Node.js - Link: https://nodejs.org/en/download/

    Dentro do Node.js já está incluso o npm que é um gerenciador de pacote para desenvolvimento java. No npm, por sua vez, também vem incluso o npx que é cli para geerenciamento de pacotes mobile.

Comandos do terminal

Comandos Resultado
npm init Cria um packge json. Um arquivo Json onde são descritas as necessidades do projeto.
cat packge.json Recupera as informações do pacote json.
npm install autocomplete Instala uma determinada biblioteca. Neste exemplo uma biblioteca para completar os códigos digitados.
npx create-react-app hello-word Cria o projeto já com as bibliotecas necessários para rodar o React.
